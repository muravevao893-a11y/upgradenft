# UPGRADE NFT Mini App

Telegram Mini App для бота с экранами:
- Задания
- Апгрейды
- Кейсы
- Бонусы
- Вы (профиль)

## Реализовано

- Компактный черно-белый интерфейс с нижней навигацией.
- Апгрейд-механика:
  - блок `Ваши NFT` и `Хочу получить`;
  - расчет шанса по стоимости и tier;
  - круг шанса (заполнение + орбитальная стрелка).
- Профиль в стиле short-video приложений:
  - данные Telegram-пользователя;
  - статы;
  - вкладки `Мои NFT` и `Выбитые NFT`.
- Подключение кошелька TON через TonConnect UI (`Connect TON Wallet` в профиле).

## Файлы TonConnect

- `tonconnect-manifest.json`
- `icon-180.png`

Если домен/репозиторий отличается, обнови URL в `tonconnect-manifest.json`.

## Запуск

1. Подними статический сервер в папке проекта.
2. Для полного функционала запускай из Telegram Mini App (в браузере будет fallback-режим гостя).

## Продакшен

- На сервере обязательно валидируй `Telegram WebApp initData`.
- Не храни критичную логику шансов только на фронте.
