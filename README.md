# UPGRADE NFT Mini App

Мини-приложение для Telegram-бота с нижней навигацией и экранами:
- Задания
- Апгрейды
- Кейсы
- Бонусы
- Вы (профиль из Telegram)

## Локальный запуск

1. Открой `index.html` в браузере или подними любой статический сервер.
2. Для корректной работы Telegram-профиля запускай в Telegram Mini App (в обычном браузере покажется гостевой профиль).

## Что уже реализовано

- Мобильная адаптивная верстка.
- Красивая нижняя панель в стиле коротких видео-приложений.
- SVG-иконки вкладок.
- Профиль пользователя через `Telegram.WebApp.initDataUnsafe.user` (без отдельной регистрации).
- Базовая механика апгрейда с шансом успеха.

## Важно для продакшена

- На сервере обязательно проверяй подпись `initData` от Telegram.
- Если нужен номер телефона, его обычно запрашивают через бота (кнопка с запросом контакта), а не из Mini App напрямую.
